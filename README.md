# 💱 Currency Converter

🖋️ *Converting values across borders — one request at a time.*

A simple and responsive currency converter built using JavaScript, fetching real-time exchange rates and providing quick, accurate conversions between currencies.

---

## 🚀 Features

* 🌐 Real-time currency conversion
* 🔄 Fetches live exchange rates via API
* 💻 Clean and responsive UI
* ⚡ Fast and lightweight
* 🧩 Easy to use and understand

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** Exchange Rate API
* **Version Control:** Git & GitHub

---

## ⚙️ How It Works

1. Select base and target currencies
2. Enter the amount
3. Fetches live exchange rates from API
4. Displays converted value instantly

---

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Lekhak666/CurrencyConverter.git
```

Navigate to the project folder:

```bash
cd CurrencyConverter
```

Open `index.html` in your browser.
